# Release Notes v5.0.0

> **Orion v5.0.0** is here — the next generation of the AI-first design system. This release consolidates years of improvements into a cleaner, more powerful package.

**Release Date**: March 2026
**Status**: Production Ready

---

## What Changed in v5.0.0

### Major Changes

#### 1. Consolidated @orion-ds/core into @orion-ds/react

The `@orion-ds/core` package has been merged into `@orion-ds/react`. This means:

- ✅ **Single package install** — One npm install instead of two
- ✅ **Simpler imports** — All components and tokens come from `@orion-ds/react`
- ✅ **Smaller bundle** — Consolidated build, no duplicate code
- ✅ **Better maintenance** — One package to update, not two

**What you need to do:**
- Remove `@orion-ds/core` from `package.json`
- Update CSS import to `import '@orion-ds/react/styles.css'`
- See the [Migration Guide](#migration-guide) below for step-by-step instructions

#### 2. Auto-Generated TypeScript Types

All TypeScript types are now auto-generated from JSON tokens. This ensures types always match your design tokens and eliminates drift between code and design.

**What changed:**
- Types are 100% generated from `tokens/primary.json`, `tokens/light.json`, `tokens/dark.json`
- No manual type updates needed — if you change a token, types update automatically
- Import types from `@orion-ds/react` with full autocomplete

#### 3. ThemeProvider Now Includes Font Loading

Fonts load automatically via `ThemeProvider`. No manual Google Fonts setup needed.

**Before v5:**
```tsx
// Had to manually load fonts
<FontLoader>
  <ThemeProvider>
    <App />
  </ThemeProvider>
</FontLoader>
```

**After v5:**
```tsx
// Fonts load automatically
<ThemeProvider>
  <App />
</ThemeProvider>
```

#### 4. MCP Server Included (AI Integration)

Orion now ships with an MCP (Model Context Protocol) server for AI agents. Use with Claude Code, Cursor, Cline, and other AI tools to generate Orion-compliant code automatically.

**Setup (3 steps):**
```bash
npm install @orion-ds/mcp
```

Then configure in your tool settings. See [AI Integration](/docs/ai-integration) for details.

## Breaking Changes

If you're upgrading from v4, read about breaking changes:

| Change | Action Required |
|--------|-----------------|
| `@orion-ds/core` removed | Remove from `package.json`, update imports |
| CSS import path changed | Use `@orion-ds/react/styles.css` |
| `@orion-ds/react/client` entry point | Recommended for SSR (Next.js App Router) |
| Types are auto-generated | No manual type maintenance needed |

**⚠️ Important:** TypeScript types have changed. Run `npm install @orion-ds/react@latest` and verify no `@orion-ds/core` imports remain.

## Migration Guide

Ready to upgrade? See the complete [Migration Guide from v4 to v5](/docs/migration-v5).

**Quick steps:**
1. Update `package.json`: `npm install @orion-ds/react@latest`
2. Remove `@orion-ds/core` from `package.json`
3. Update CSS import to `import '@orion-ds/react/styles.css'`
4. Run tests to verify everything works

Takes about 30 minutes for most projects.
