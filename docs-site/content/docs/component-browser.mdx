## Pro Tips

### Customization Patterns

**Theming**
All components automatically adapt to your theme and brand:
```tsx
import { ThemeProvider } from '@orion-ds/react';

<ThemeProvider>
  {/* All components inherit theme/brand */}
  <Button>Click me</Button>
</ThemeProvider>
```

**Responsive Sizing**
Components automatically scale based on the current mode (Display, Product, App):
```tsx
<Button size="md">  {/* Adjusts to mode context */}
  Submit
</Button>
```

**Icon Integration**
All components support Lucide icons:
```tsx
import { Download } from 'lucide-react';

<Button icon={<Download size={20} />}>
  Download
</Button>
```

### Component Composition

**Combine related components** for richer UIs:
```tsx
// Form workflow
<Card>
  <Card.Header>Login</Card.Header>
  <Card.Body>
    <Field label="Email" type="email" />
    <Field label="Password" type="password" />
    <Button fullWidth variant="primary">Sign In</Button>
  </Card.Body>
</Card>
```

**Use layout components** to organize:
```tsx
import { Stack, Divider, Card } from '@orion-ds/react';

<Stack gap="spacious">
  <Card>Profile</Card>
  <Divider />
  <Card>Settings</Card>
</Stack>
```

**Nest navigational components**:
```tsx
<Navbar>
  <Navbar.Brand>My App</Navbar.Brand>
  <Tabs defaultTab="home" tabs={[
    { id: 'home', label: 'Home', content: <Home /> },
    { id: 'about', label: 'About', content: <About /> },
  ]} />
</Navbar>
```

### Common Patterns

**Modal Confirmation**
```tsx
const [open, setOpen] = useState(false);

<Modal open={open} onOpenChange={setOpen}>
  <Modal.Header>Delete Item?</Modal.Header>
  <Modal.Body>This action cannot be undone.</Modal.Body>
  <Modal.Footer>
    <Button variant="ghost" onClick={() => setOpen(false)}>Cancel</Button>
    <Button variant="danger" onClick={handleDelete}>Delete</Button>
  </Modal.Footer>
</Modal>
```

**Form Validation**
```tsx
const [errors, setErrors] = useState({});

<Field
  label="Email"
  error={errors.email}
  onChange={(e) => validate(e.target.value)}
/>
```

**Loading States**
```tsx
<Button isLoading={isSubmitting}>
  {isSubmitting ? 'Saving...' : 'Save'}
</Button>
```

**Empty States**
```tsx
{items.length === 0 ? (
  <EmptyState
    title="No items yet"
    description="Create your first item to get started"
    action={<Button>Create Item</Button>}
  />
) : (
  <List items={items} />
)}
```

## Troubleshooting

### "I can't find a component for my use case"

1. **Check the search** — Use the component search above
2. **Browse by category** — Look through all categories
3. **Check related components** — Components often can be combined
4. **Request a component** — [GitHub Issues](https://github.com/orion-ds/orion/issues)

### "A component doesn't look right"

1. **Verify CSS import** — `import '@orion-ds/react/styles.css'`
2. **Check ThemeProvider** — Wrap your app with `<ThemeProvider>`
3. **Check variants** — Most components have multiple variants
4. **Check responsive** — Size adjusts based on window size and mode

### "I want to customize a component"

1. **Use CSS variables** — Override theme tokens
2. **Use component props** — Most customization is in props
3. **Use CSS Modules** — Create custom styling
4. **Check the guide** — [Customization Guide](/docs/customization)
