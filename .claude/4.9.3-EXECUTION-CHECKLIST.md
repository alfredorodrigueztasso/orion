### Step 1.2: Understand the Bug
**Location**: `packages/react/tsconfig.build.json`
**Problem**: Includes `.stories.tsx` files in TypeScript compilation
**Effect**: Causes `tsc` to fail on self-imports of @orion-ds/react package
**Result**: Vite never runs → only theme.css generated → broken release

**Current content** (PROBLEMATIC):
```json

  "include": [
    "src/**/*.ts",

    "**/*.test.tsx",
```

## Troubleshooting Guide

### Issue: `tsc` still fails after fix
**Diagnosis**: Check for other files importing @orion-ds/react
```bash
grep -r "@orion-ds/react" packages/react/src --include="*.tsx" --include="*.ts" | grep -v node_modules
```
**Fix**: Move those imports to stories only, or fix the circular reference

### Issue: dist/ still incomplete after build
**Diagnosis**: Check vite build output for errors
```bash
cd packages/react && npm run build 2>&1 | tail -50
```
**Fix**: Address vite errors (usually CSS or asset issues)

### Issue: validate-dist-completeness script fails
**Diagnosis**: Check which files are missing
```bash
node packages/react/scripts/validate-dist-completeness.js
```
**Fix**: Manually rebuild until all exports present

### Issue: npm publish fails
**Diagnosis**: Check npm login status
```bash
npm whoami
# If error, login again
npm login
```
**Fix**: Login to npm, then retry publish

### Issue: Deprecation doesn't work
**Diagnosis**: May need to wait for npm cache to clear
```bash
# Wait 5 minutes
npm cache clean --force
npm info @orion-ds/react@4.9.2
```
**Fix**: Retry deprecation command
